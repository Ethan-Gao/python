1.4 数学与表达式
    a. 整数除法: 默认只取整数部分，因此1/2结果为0
    b. 小数/浮点数除法方式有两种
        b1. 分子或者分母有一个为小数/浮点数，例如1/2.0
        b2. 使用from __future__ import division，则/默认浮点，//整除
    c. 求余: %
        都为整数则结果整数，有一个小数则结果小数
    d. 幂: **
        幂运算优先级比取反优先级高，所以-3 ** 2 = - (3 ** 2)
    e. 长整型数据处理: python可自动处理非常大的数据，包括各种加减乘除运算
    f. 进制: 16进制 - 0x, 8进制 - 0
        各种进制可随意做各种运算，最后都转化为10进制

1.5 变量
    a. 可包含字母、数字、下划线，但是不能以数字开头
    b. 定义时，与C语言不同，需要写成a, b, ... =

1.7 获取用户输入
    有两个函数，input和raw_input，区别如下
    input: 仅能接受数值型输入，比如-1, 2.3, 12，返回为数字型
    raw_input: 能接受各种输入，比如数值，字符，混合等等，返回为字符型

1.8 函数
    a. 特别注意print，可当函数用，但是可以不用括号
        因此print "hello"与print("hello")作用相同
    b. abs-取绝对值，round-四舍五入

1.9 模块
    a. 导入函数，import和from ... import ...，区别如下
        import: 导入模块，按照"模块.函数"使用
        from ... import ...: 导入模块内的函数，按照"函数"使用
        因此同时from math import *和from cmath import *会冲突，有同名函数
    b. sqrt函数
        math.sqrt仅能计算实数浮点，cmath.sqrt能同时计算实数和虚数开方
    c. 虚数支持
        python默认支持，无须使用import cmath导入
    d. 函数引用
        当使用import math之后，可以直接使用变量来引用函数，比如x = math.sqrt
    e. 几个函数比较
        round-四舍五入，floor-下舍，ceil-上入
